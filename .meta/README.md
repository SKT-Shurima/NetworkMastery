# .meta — 项目内部文档

本目录包含 NetworkMastery 项目的内部工作文档，**不发布到站点**。

## 目录说明

| 目录 | 内容 |
|------|------|
| `planning/` | 内容规划与写作规范，包括 OUTLINE（大纲）、PROGRESS（进度）、MASTER_PROMPT（写作提示词）、REWRITE_PROGRESS（重写日志） |
| `reports/` | 各阶段完成报告，记录 Phase 1/2 的工作成果与验收清单 |
| `dev-guides/` | 开发者技术指南，包括主题配置、图表升级方案、Markdown 集成示例、构建修复记录 |

## 写作规范

新增文章请参考 `planning/MASTER_PROMPT.md`，遵循 5 层递进结构和 Mermaid 可视化规范。
